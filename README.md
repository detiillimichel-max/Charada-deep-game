# 🔍 O Que é, O Que é? - Jogo de Adivinhas

Um jogo interativo com 9 charadas clássicas do tipo "O que é, o que é?" para testar seu raciocínio!

## ✨ Funcionalidades

- 🎴 **9 cards** com charadas desafiadoras
- 💾 **Salvamento automático** (seu progresso é mantido no navegador)
- 🎯 **Sistema de pontuação** (1 ponto por cada resposta revelada)
- 🎨 **Design responsivo** (funciona em celulares, tablets e computadores)
- 🔄 **Botão reset** para recomeçar o jogo
- 📱 **Animações suaves** ao revelar respostas

## 🚀 Como jogar

1. Leia a charada em cada card
2. Tente adivinhar a resposta
3. **Clique no card** para revelar a resposta correta
4. Cada resposta revelada vale 1 ponto
5. Tente completar as 9 charadas!

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Animações)
- JavaScript (Vanilla - sem frameworks)
- LocalStorage para persistência de dados
